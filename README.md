## 安装

```shell
npm install ez-vue-select
# or
yarn add ez-vue-select
```

## 快速开始

```javascript
import Vue from "vue";
import EasySelect from "ez-vue-select";
import "ez-vue-select/lib/ez-vue-select.css";

Vue.use(EasySelect);
```

## 开发

### 安装依赖

```shell
yarn
```

### 启动案例

```
yarn serve
```

### 打包库

```
yarn build-lib
```
